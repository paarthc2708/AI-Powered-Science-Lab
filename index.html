Acid-Base Titration

Acid-Base Titration

Aim of the Experiment

To determine the concentration of an unknown acid solution by titrating it with a base of known concentration.

Apparatus Required

  • Burette
  • Pipette
  • Conical Flask
  • Stand

Chemicals Required

  • Acid (e.g., Hydrochloric Acid)
  • Base (e.g., Sodium Hydroxide)
  • Indicator (e.g., Phenolphthalein)

Procedure

  1. Fill the burette with the base solution.
  2. Use the pipette to transfer a measured volume of the acid into the conical flask.
  3. Add a few drops of indicator to the acid in the conical flask.
  4. Slowly add the base from the burette to the acid while swirling the flask.
  5. Stop adding the base when the indicator changes color, indicating neutralization.

Calculations

Use the volume of base added to calculate the concentration of the acid using the formula: M1V1 = M2V2, where M1 and V1 are the molarity and volume of the acid, and M2 and V2 are the molarity and volume of the base.

Observations

Record the volume of base used to reach the endpoint and any color changes observed.

Precipitation Reaction

Precipitation Reaction

Aim of the Experiment

To observe the formation of an insoluble solid (precipitate) when two aqueous solutions react with each other.

Apparatus Required

  • Test Tube
  • Test Tube Stand
  • Dropper
  • Beaker

Chemicals Required

  • Salt Solution A (e.g., Silver Nitrate solution)
  • Salt Solution B (e.g., Sodium Chloride solution)

Procedure

  1. Take a clean test tube and add Salt Solution A into it.
  2. Using a dropper, slowly add Salt Solution B to the same test tube.
  3. Mix the solutions gently by swirling the test tube.
  4. Observe the formation of a precipitate in the test tube.

Chemical Equation

Salt Solution A + Salt Solution B → Insoluble Salt (precipitate) + By-product
AgNO3 (aq) + NaCl (aq) → AgCl (s) + NaNO3 (aq)

Observations

A white insoluble solid forms immediately after mixing the two solutions.
The solution becomes cloudy due to the formation of a precipitate.

pH Indicator Experiment

pH Indicator Experiment

Aim of the Experiment

To identify whether a given solution is acidic, basic or neutral using a pH indicator.

Apparatus Required

  • Test Tube
  • Test Tube Stand
  • Dropper
  • Beaker

Chemicals Required

  • Test solution (acidic / basic / neutral)
  • pH Indicator (e.g., universal indicator or litmus solution)

Procedure

  1. Take a clean test tube and add a small amount of test solution into it.
  2. Using a dropper, add a few drops of pH indicator to the test solution.
  3. Mix the solution gently by swirling the test tube.
  4. Observe the color change in the solution.
  5. Compare the color with the pH color chart to determine whether the solution is acidic, basic or neutral.

Chemical Equation

Test Solution + pH Indicator → Color Change
HCl (aq) + Universal Indicator (aq) → Color Change

Observations

A color change occurs in the solution.
The solution changes color depending on its acidity or basicity.

  • Red / Orange → Acidic
  • Green → Neutral
  • Blue / Purple → Basic
